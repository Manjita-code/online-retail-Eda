PROJECT TITLE:
Part B – Exploratory Data Analysis (EDA)

DESCRIPTION:
This project performs Exploratory Data Analysis (EDA) on an online retail dataset to identify
patterns, trends, missing values, anomalies, and regional insights.

FILES INCLUDED:

1. eda\_analysis.ipynb – EDA code
2. online\_retail.csv – Dataset
3. requirements.txt – Required Python libraries
4. README.md – Instructions to run the project

HOW TO RUN THE PROJECT:

Step 1: Install Python 3 on your system

Step 2: Open Command Prompt and go to project folder
cd EDA\_Project

Step 3: Install required libraries
pip install -r requirements.txt

Step 4: Run the EDA notebook
jupyter notebook eda\_analysis.ipynb

TOOLS USED:
Python, Pandas, NumPy, Matplotlib, Jupyter Notebook
